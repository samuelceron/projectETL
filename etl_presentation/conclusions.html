Conclusions


The main benefits regarding ETL process are:

  • Information clarity: Avoiding unclear raw data
  • Information completeness: There are not missing pieces
  • Information quality: Giving confidence to use data
  • Information velocity: New data can be added in to the DB faster

A key starting point is Data Modeling, based on the Entity Relationship Diagram

If you have a lists of attributes inside a column, you should consider a document database like MongoDB (no relational data management system).

Transformation process, particularly NORMALIZATION, spends 80% of our time in the project
